Velkommen til ID Psykoterapeutisk Forening

ID Psykoterapeutisk Forening er et netværk for certificerede ID Psykoterapeuter i Danmark.

På vores hjemmeside kan du læse om ID Psykoterapi, og du kan se profilerne på de ID Psykoterapeuter, som er medlemmer i foreningen. Vi har også et nyhedsbrev, som du kan tilmelde dig og høre nyt om kurser m.v.

Vi formidler viden og informationer, der kan inspirere til større forståelse for, hvad psykoterapi er, og hvad psykoterapi kan bruges til.

Hvis du har noget, du vil spørge om, eller noget du gerne vil kommentere, så er du velkommen til at kontakte os: piabalk@gmail.com
